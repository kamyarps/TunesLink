# Support

For setup and connection problems, start with the [README troubleshooting guide](README.md#troubleshooting).
When reporting a bug, include the TunesLink version, Android and Windows versions, the exact step
that failed, and whether manual address entry works. The Windows tray menu can open the diagnostics
folder; its bounded log contains event codes and exception types, not tokens, addresses, or music
metadata.

Never post pairing codes, bearer tokens, signing material, personal IP addresses, private crash
dumps, or music-library data. Report security issues through the private process in
[SECURITY.md](SECURITY.md), not a public issue.
